# Expense Tracker Dashboard

Responsive Expense Management Dashboard built using React, Context API, Tailwind CSS, and Recharts featuring CRUD transaction management, interactive analytics visualization, dark mode support, and localStorage persistence.

---

## Features

- Add income and expense transactions
- Update existing transactions
- Delete transactions
- Real-time balance calculation
- Interactive Expense vs Balance pie chart
- Dark / Light theme support
- LocalStorage persistence
- Responsive dashboard UI
- Context API state management

---

## Tech Stack

- React.js
- Context API
- Tailwind CSS
- Recharts
- SweetAlert2
- Vite

---

## Installation

Clone repository:

```bash
git clone https://github.com/RakshatTiwari/expense-tracker-dashboard.git
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
